# Contributing

感谢你参与 AGENTSMD。

## 提交前要求

1. 遵循 `AGENTS.md` 的模式与工作流。
2. 任何部门变更先改模板/索引，再改条目。
3. 受保护文件变更必须在 PR 描述中说明理由。
4. 本地执行以下命令并通过：

```bash
bash scripts/md_sync.sh
```

## 提交规范

- 使用清晰的提交信息，建议前缀：
  - `docs:` 文档规则/条目变更
  - `feat:` 新能力
  - `fix:` 修复
  - `chore:` 维护性调整

## PR 要求

- 描述问题、改动范围、影响部门。
- 若涉及规则变化，附上迁移说明。
- 若涉及 UI，附截图（可选）。
