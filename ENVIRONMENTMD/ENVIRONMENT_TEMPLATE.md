# ENVIRONMENT_TEMPLATE

## 记录规则

* 同一 `Key` 仅保留一个当前有效条目。
* `Key` 已存在时更新该条目并刷新文件名时间戳。
* `Key` 不存在时新建条目。

## Metadata

* **修改者：** [执行器Agent的名称]
* **Key：** [环境主题名称]

## Summary

* [总结此环境事实，必须一句话，单行输出]

## Thought

* [记录环境信息的背景原因，可以多条输出，每条一句话]
* [环境风险或兼容性判断，可以多条输出，每条一句话]
* [其他必须记录的环境思考，可以多条输出，每条一句话]

## Action

* [本次环境采集或确认动作，可以多条输出，每条一句话]
* [本次涉及的命令或工具，可以多条输出，每条一句话]
* [其他必须记录的执行细节，可以多条输出，每条一句话]
