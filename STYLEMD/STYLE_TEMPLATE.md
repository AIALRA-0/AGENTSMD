# STYLE_TEMPLATE

## 记录规则

* STYLEMD 为条目模式（entry），以文件后缀为单位维护风格规范。
* 同一 `Key` 仅保留一个当前有效条目。
* `Key` 已存在时更新条目并刷新文件名时间戳；`Key` 不存在时新建条目。
