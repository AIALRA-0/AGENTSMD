# TOOL_TEMPLATE

## 记录规则

* TOOLMD 为条目模式（entry），按工具主题 `Key` 维护本地工具清单。
* 同一 `Key` 仅保留一个当前有效条目。
* `Key` 已存在时更新条目并刷新文件名时间戳；`Key` 不存在时新建条目。
