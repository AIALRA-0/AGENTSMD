# RESOURCE_TEMPLATE

## 使用说明

* RESOURCEMD 为条目模式（entry），用于登记资源路径，不存放资源正文。
* 路径规则：资源已下载到本地时写绝对路径；未下载时写 URL。
* 同一 `Key` 只维护一个当前有效条目，更新时刷新文件名时间戳。

## Metadata

* **修改者：** [执行器Agent的名称]
* **Type：** [DOCS / PATENT / REPO / API / DATASET / OTHER]
* **Key：** [资源主题唯一键]
