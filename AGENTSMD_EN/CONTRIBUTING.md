# Contributing

thank you for participating AGENTSMD。

## Presubmission requirements

1. follow `AGENTS.md` patterns and workflows。
2. Modify the template first before any department changes/Index，Change the entry again。
3. Changes to protected files must be PR Explain the reason in the description。
4. Execute the following command locally and pass：

```bash
bash scripts/md_sync.sh
```

## Submission specifications

- Use clear commit messages，Suggested prefix：
  - `docs:` Document rules/Entry changes
  - `feat:` new capabilities
  - `fix:` Repair
  - `chore:` maintenance adjustments

## PR request

- Describe the problem、Change scope、Influence departments。
- In case of rule changes，Attached are migration instructions。
- If involved UI，Attached screenshot（Optional）。
